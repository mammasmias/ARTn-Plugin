*ninit*
======================

Syntax
""""""

.. parsed-literal::

   ninit = arg

* arg = integer


Default
"""""""

.. code-block:: bash

   ninit = 3


Description
"""""""""""
Specify the minimal number of pushes with the initial push vector. Value 0 means go directly to Lanczos.
