*forc_thr*
======================

Syntax
""""""

.. parsed-literal::

   forc_thr = arg

* arg = real


Default
"""""""

.. code-block:: bash

   forc_thr = 1e-3


Description
"""""""""""

The configuration has converged to either a saddle point, or a minimum, when the sum of the parallel and perpendicular components of the atomic forces is lower than ``forc_thr``.
